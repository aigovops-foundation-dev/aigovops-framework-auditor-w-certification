What's in the box

Five components, one repo, Apache-2.0 throughout.

Web app REACT

React + Vite frontend. Submit a review, watch agents work, and see HITL queues, attestations, and conformance scoring in real time.

Edge functions DENO

The agent pipeline, signed audit chain, attestation issuance, and verification endpoint. All hash-pinned in the canary manifest.

AOS spec YAML

Machine-readable AI Operating Standard v0.1 — controls, frameworks, evidence expectations. Vendor-neutral and forkable.

@aigovops/verify CLI

npm package that re-verifies any attestation PDF or audit chain offline. Lets anyone math-check our claims without trusting our server.

Canary manifest CI

SHA-256 manifest of every governance-critical file. A weekly GitHub Action opens an issue if anything drifts. Public mirror at /canary-manifest.json.

QAGA program DOCS

Qualified AI Governance Assessor charter, firm independence rules, and the public registry. Foundation-stewarded, not vendor-locked.

60-second quick start

Clone, install, run. Zero config.

# 1. Clone
git clone https://github.com/aigovops/aigovops-framework.git
cd aigovops-framework

# 2. Install
npm install

# 3. Run
npm run dev

# 4. Verify any attestation, anywhere, offline
npx @aigovops/verify pdf ./attestation.pdf
